コンソールでやる or SQLでやる
## confirmed_at設定
```sql
UPDATE 
	users 
SET 
	(
		confirmed_at = current_timestamp,
		encrypted_password = hashed_password
	)
WHERE 
	email = "tsubaki.toshio@eneos.com"
;

SELECT id, confirmed_at FROM users WHERE email = "tsubaki.toshio@eneos.com";
```

## コンソール
```ruby
> require "bcrypt"
> hashed_password = BCrypt::Password.create("Eneos-123456")

> sql = ActiveRecord::Base.sanitize_sql_array(["UPDATE users SET confirmed_at = ?, encrypted_password = ? WHERE email = 'tsubaki.toshio@eneos.com'", Time.now, hashed_password])
> ActiveRecord::Base.connection.execute(sql)
```
