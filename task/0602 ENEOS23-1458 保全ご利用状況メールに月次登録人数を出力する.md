# 先月の集計
```sql
SELECT
  DATE_FORMAT(CONVERT_TZ(created_at, 'UTC', 'Asia/Tokyo'), '%Y%m') AS `month`,
  COUNT(id) AS count
FROM
  users
WHERE
  valqua <> 1
  AND DATE_FORMAT(CONVERT_TZ(created_at, 'UTC', 'Asia/Tokyo'), '%Y%m') = DATE_FORMAT(CONVERT_TZ(CURRENT_TIMESTAMP, 'UTC', 'Asia/Tokyo') - INTERVAL 1 MONTH, '%Y%m')
GROUP BY
  `month`
;
```
